
## 1.用户 

### 1.1 用户登录

###### 说明:
不需要token

###### 请求方式

POST

###### 格式

json

###### 请求url

/api/users/login

###### 参数

| 参数名       | 必选 |  类型  |   说明   |
| ------------ | :--: | :----: | :------: |
| userAccount  |  是  | string | 用户账号 |
| userPassword |  是  | string |   密码   |

###### 获取

```json
{
    "userAccount":"5170",
    "userPassword":"5170"
}
```

###### 返回

返回的header中包含了token，登录之后的所有请求的请求头都需要token，否则请求会被拦截
```json
{
    "code":200,
    "message":"登录成功",
    "data":{
        "userId":"12020051415894267923390001"
    }
}
```

###### 返回参数说明

| 参数名 |  类型  |  说明  |
| ------ | :----: | :----: |
| userId | string | 用户id |

